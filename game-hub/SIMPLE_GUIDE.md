# 🎮 游戏中心 - 简单使用指南

## 方法一：局域网共享（推荐）

### 步骤：
1. **双击运行** `share-local.bat`
2. **等待服务器启动**（看到 "Server running at" 信息）
3. **获取访问地址**：
   - 你自己的电脑：`http://localhost:3000`
   - 朋友电脑：`http://你的IP地址:3000`

### 如何找到你的IP地址：
- 运行 `share-local.bat` 后会自动显示
- 或者按 `Win+R` 输入 `cmd`，然后输入 `ipconfig`
- 找到 "IPv4 地址"，通常是 `192.168.x.x`

### 分享给朋友：
- 确保朋友和你连接同一个WiFi
- 把 `http://你的IP地址:3000` 发给朋友
- 朋友在浏览器打开即可

## 方法二：便携版分享

### 步骤：
1. **双击运行** `create-portable.bat`
2. **等待构建完成**
3. **找到生成的文件夹**：`game-hub-portable`
4. **压缩这个文件夹**为 zip 文件
5. **发送给朋友**

### 朋友使用方法：
1. 解压 zip 文件
2. 双击 `index.html` 文件
3. 选择用浏览器打开

## 方法三：直接文件分享

### 最简单的方法：
1. **直接发送整个项目文件夹**给朋友
2. 朋友需要安装 Node.js（可选）
3. 朋友双击 `start.bat` 即可运行

## 📋 快速开始（推荐方法一）

**立即开始**：双击 `share-local.bat`，然后分享链接给朋友！

## 💡 小贴士

- **方法一**最适合临时和朋友一起玩
- **方法二**适合发给没有网络的朋友
- **方法三**最简单，但需要朋友有基本电脑知识

## 🔧 故障排除

如果遇到问题：
1. 确保防火墙允许访问端口3000
2. 检查是否在同一个网络
3. 重启脚本试试
