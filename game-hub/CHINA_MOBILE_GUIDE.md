# 游戏中心 - 国内手机访问指南

## 📱 针对国内用户的解决方案

由于你的朋友在中国国内，无法访问外网服务（需要VPN），我们提供以下**纯国内网络环境**的解决方案：

## 🎯 推荐方案：局域网共享（最稳定）

### 步骤1：确保同一网络环境
- 你和朋友连接**同一WiFi网络**（家庭、公司、学校等）
- 或者使用手机热点共享网络

### 步骤2：运行一键脚本
双击运行：`china-mobile.bat`

### 步骤3：获取访问地址
脚本会显示类似：
```
http://192.168.1.100:3000
```

### 步骤4：朋友手机访问
- 朋友手机连接同一WiFi
- 手机浏览器输入上面显示的IP地址
- 即可访问游戏中心

## 🔄 备选方案：文件传输

### 方案A：微信文件传输
1. 运行：`create-portable.bat` 创建便携版
2. 将生成的 `game-hub-portable.zip` 通过微信发送给朋友
3. 朋友解压后双击 `start.bat` 即可本地运行

### 方案B：国内云盘分享
1. 使用百度网盘、阿里云盘等国内服务
2. 上传整个项目文件夹或便携版压缩包
3. 分享链接给朋友下载

## 🛠️ 常见问题解决

### 问题1：手机无法访问IP地址
**解决方案：**
1. 检查防火墙设置
2. 临时关闭Windows防火墙测试
3. 确保Node.js被允许通过防火墙

### 问题2：不在同一网络
**解决方案：**
1. 使用手机热点创建共享网络
2. 电脑连接朋友手机的热点
3. 重新运行脚本获取新IP

### 问题3：朋友想离线使用
**解决方案：**
1. 使用便携版方案
2. 朋友下载后完全离线运行
3. 无需网络连接

## 📋 快速开始步骤

### 最简单的方法（推荐）：
1. **双击** `china-mobile.bat`
2. **复制**显示的IP地址（如：http://192.168.1.100:3000）
3. **发给朋友**，确保连接同一WiFi
4. **朋友手机浏览器**输入地址访问

### 文件传输方法：
1. **双击** `create-portable.bat`
2. **等待**生成便携版压缩包
3. **通过微信**发送给朋友
4. **朋友解压**后双击 `start.bat`

## 💡 重要提示

- ✅ **局域网方案**：最稳定，实时更新
- ✅ **便携版方案**：离线可用，无需网络
- ❌ **避免使用**：需要外网的内网穿透服务
- 🔒 **安全性**：局域网访问很安全

## 🆘 故障排除

### 如果无法访问：
1. 检查电脑和手机是否连接同一WiFi
2. 尝试关闭防火墙临时测试
3. 重启路由器重新获取IP
4. 使用手机热点创建共享网络

### 技术支持：
- 优先使用局域网共享方案
- 备选使用便携版文件传输
- 两种方案都完全在国内网络环境下可用

---

**总结**：对于国内用户，最推荐**局域网共享**方案，简单稳定且无需任何外网服务！
